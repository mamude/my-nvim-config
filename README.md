# my-nvim-config

This is my personal Neovim configuration, you can use for Python, Go and NodeJs.

## How to install
**Neovim**

    curl -LO https://github.com/neovim/neovim/releases/latest/download/nvim-linux64.tar.gz
    sudo rm -rf /opt/nvim
    sudo tar -C /opt -xzf nvim-linux64.tar.gz
